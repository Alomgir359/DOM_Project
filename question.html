Question-1: What are the different ways to select an element in the DOM?

Answer-1: The different ways to select an element in the DOM are getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), and querySelectorAll().

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer-2: innerHTML returns or sets HTML content, innerText returns only visible text, and textContent returns all text, including hidden ones.

Question-3: What is event delegation in the DOM?

Answer-3: Event delegation is a technique where a parent element handles events for its child elements using event bubbling, improving performance.

Question-4: What is event bubbling in the DOM?

Answer-4: Event bubbling is the process where an event starts from the target element and propagates up to its ancestors in the DOM hierarchy.
